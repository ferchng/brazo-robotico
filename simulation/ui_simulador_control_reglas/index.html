Simulador + Control con Reglas

La simulación es la autoridad para M2/M3/M4. Al conectarse, la UI lee la postura real del brazo y actualiza sliders + simulación con esa pose.
Sin conectar Cola: 0
Simulación
Muestra base, L1, offset M3, L2, L3, bloque de garra y validación de la pose.
Control
Los sliders actualizan la simulación al instante. Si una pose es inválida, el control vuelve al último valor válido. La sincronización desde hardware refleja la pose real aunque sea inválida.
Consola
Para debug manual: posall, read 3, scan.
