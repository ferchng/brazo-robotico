Control de Servos STS3215

Interfaz local por Web Serial. Mueve un servo por vez usando el protocolo del sketch del ESP32.
Sin conectar
Seguridad
Esta UI envia comandos de a uno: <id> <grados>. Respeta tus limites por motor.
Para el motor 6 la UI usa rango logico 0..140. El offset +10 grados lo sigue aplicando tu sketch del ESP32.
Cooldown de movimientos: 1 segundo entre comandos de movimiento.
Consola de comandos
Sirve para debug: posall, read 3, scan, o comandos de movimiento.
